# LIS 2026 – Lot Information System

Lot Information System. Built with Next.js (frontend) and FastAPI (backend)

## Repository layout

- `frontend/` – Next.js 14+ (App Router), Tailwind CSS
- `backend/` – FastAPI (from MVP 2)

## Prerequisites

- Node.js 18+
- npm (or yarn/pnpm)

## Run the frontend (MVP 1)

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). No backend required for MVP 1.

## Build for production

```bash
cd frontend
npm run build
npm start
```

## Environment

Copy `.env.example` to `.env` or `.env.local` and set variables as needed. MVP 1 runs without backend or SSO.
