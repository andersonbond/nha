# CIIM

Commercial, Institutional, Industrial, and Mixed-use units.

## Description

CIIM manages non-residential or mixed-use units within projects: commercial, institutional, industrial, and mixed-use. Each unit has a code, name, type, location (region, project), total value, area, and status (e.g. leased, operational, available).

## Features

- **List CIIM units** – Table of units with unit code, name, type, region, project, total value, created date.
- **Filter** – Modal filter by unit code, unit name, unit type (Commercial / Institutional / Industrial / Mixed-use), region, and project.
- **Add unit** – Button placeholder; full create flow planned for MVP 3.
- **Quick insights** – Unit count, total value, breakdown by type, number of projects.
- **Edit / Delete** – Placeholder actions (Coming in MVP 3).
