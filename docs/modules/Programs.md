# Programs

Manage housing programs (financing and terms).

## Description

The Programs module defines housing programs with reference codes (e.g. MC ref), interest rate, delinquency rate, max term in years, and approval status. Projects are linked to a program for financing and terms.

## Features

- **List programs** – View all programs in a table with filters (MC ref, minimum interest rate, max term years).
- **Filter** – Modal filter by MC ref, interest rate (min), and max term years.
- **Add program** – Create a new program via slide-over form; optional review step before submit.
- **Edit program** – Update an existing program (program ID is read-only).
- **Delete program** – Remove a program with confirmation.
- **Quick insights** – Summary stats from the program list (e.g. counts, terms).
