# Settings

Application preferences and configuration.

## Description

Settings centralizes user and system preferences: address configuration, user profile, Lot Information System options, and appearance (theme, locale). Changes apply to the current user or the application behavior.

## Features

- **Address configuration** – Manage regions, provinces, municipalities, and barangays (e.g. for dropdowns and validation). Optional warm-up for address cache.
- **User settings** – User profile and account-related options (placeholder or minimal in current scope).
- **Lot Information System** – LIS-specific options (placeholder or minimal).
- **Appearance** – Theme (Light / Dark) and language/locale (planned; default English).
