# Reports

Generate and download reports.

## Description

The Reports module provides predefined report types that can be filtered and exported (e.g. to Excel). Reports cover applications, lots, awards, program summaries, beneficiary demographics, landholding assessment, and financial data.

## Features

- **Report types (placeholder)**  
  - Applications  
  - Lots  
  - Awards  
  - Monthly Program Summary  
  - Beneficiary Demographics  
  - Landholding Assessment  
  - Quarterly Financial Report  
- **Per-report** – Section with description, filter placeholders (date range, project/status), and **Download (xlsx)** button.
- **Planned (MVP 3)** – Actual report generation and file download; filters and preview.
