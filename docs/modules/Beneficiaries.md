# Beneficiaries

Manage beneficiary records linked to applications and programs.

## Description

The Beneficiaries module holds beneficiary information (BIN, application link, name, demographics, address, membership and category codes). Beneficiaries are often created from or linked to applications and can be associated with lot awards.

## Features

- **List beneficiaries** – View all beneficiaries in a table (BIN, name, application ID, category, address, etc.).
- **Add beneficiary** – Create a new beneficiary via a slide-over form (BIN, app ID, name, birth date, sex, civil status, address, region/province/municipality/barangay, membership/common codes, category).
- **Edit beneficiary** – Update an existing beneficiary from the same form.
- **Delete beneficiary** – Remove a beneficiary with confirmation.
- **Quick insights** – Summary stats from the beneficiary list (e.g. counts by category or region).
