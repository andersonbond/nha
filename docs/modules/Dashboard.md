# Dashboard

Brief overview of the Lot Information System (LIS) and key metrics.

## Description

The Dashboard is the home view after login. It provides a high-level snapshot of applications, beneficiaries, and lot-related data so users can see system activity at a glance.

## Features

- **Stats summary** – Counts or totals for applications, beneficiaries, projects, and lots.
- **Breakdown** – Visual breakdown (e.g. by program or status) for quick insight.
- **Recent activity** – List of recent actions or updates in the system.
