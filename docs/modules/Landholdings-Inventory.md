# Landholdings Inventory

Inventory of landholdings by owner, location, and classification.

## Description

The Landholdings Inventory module tracks land parcels: parcel number, owner, location (region, province, municipality, barangay), classification (e.g. residential, agricultural, commercial, institutional, mixed), area (hectares), and status. Used for planning and reporting on land assets.

## Features

- **List landholdings** – Table of parcels with parcel no., owner, region, province, classification, area (ha), status, created date.
- **Filter** – Modal filter by parcel no., owner, classification, region, and province.
- **Add landholding** – Button placeholder; full create flow planned for MVP 2.
- **Quick insights** – Landholding count, total area (ha), breakdown by classification, number of regions.
- **Edit / Delete** – Placeholder actions (Coming in MVP 2).
